"Il-Luzzu" is an eye-catching boat that is brightly painted in shades of brownish-red, yellow and blue. A pair of Osiris eyes are usually painted at the bow. About Luzzu

Luzzu is a generic Quality Assessment framework for Linked Open Datasets. The purpose of the Quality Framework is to provide an integrated platform that:

  1. assesses Linked Open Data (via RDF Dumps or SPARQL endpoints) in a scalable manner;
  2. provides queryable quality metadata on the assessed datasets;
  3. provides visualisations on the quality metadata.
Furthermore, we aim to create an infrastructure and a platform that (i) can be easily extensible by different third parties by creating their custom and more specific pluggable metrics required to assess their particular dataset domain, and (ii) having the the necessary ontology framework to represent the metadata about the quality of the assessed linked datasets.

The core assessment framework is make up of four modules:
  • Semantic Schema Layer;
  • Processing Unit;
  • Semantic Annotation Unit;
  • Quality Assessment Layer.
Luzzu also provides a RESTful API that allows the use of the Quality Framework from the World Wide Web.

Semantic Schema Layer

The Quality Framework is based on semantic technologies and thus has an underlying semantic vocabulary layer which currently consists of two ontologies: (i) the Dataset Quality Ontology; and (ii) the Quality Problem Report Ontology. The former describes the quality metadata representation whilst the latter describes quality problems found in the dataset itself.

Processing Unit

The Processing Unit is an integral part of the Quality Framework. In this framework release, we provide a sequential stream processor. This processor streams triples from RDF data dumps in a sequential fashion, ensuring that the framework does not suffer from memory problems when assessing large datasets. In future releases, we aim to provide a SPARQL processor that allows the framework to assess quality on data that is available only in SPARQL endpoints

Semantic Annotation Unit

The Semantic Annotation Unit generates quality metadata for the assessed dataset.

Quality Assessment Layer

The Quality Assessment Layer is the most important layer of the Quality Framework. The framework can be extended by any third party providing their own custom specific metric. The Quality Assessment Layer provides two interfaces and an abstract class that facilitate the quality framework to be a pluggable and extensible platform. More information about how metrics can be implemented can be found in the How-To section.
