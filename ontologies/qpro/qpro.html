Abstract

The Quality Problem Report (QPRO) enables the fine-grained description of quality problems found while assessing a dataset.

Term Description

In this section, the classes and properties that provide a basis for the Quality Problem Report (QPRO) are described

Represents an Exception that causes the dataset to have Quality Problems. This is an abstract class
