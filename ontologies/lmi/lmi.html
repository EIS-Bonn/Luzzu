Abstract

The Luzzu Metric Implementation ontology (LMI) enables the definition of specific metrics to be connected to their Java Implementations. This vocabulary is used in Luzzu to load the required metrics in the framework.

Term Description

In this section, the classes and properties that provide a basis for the Luzzu Metric Implementation ontology (LMI) are described

Classes

Metric Configuration

(Used for API Call) An instance of this class should have one or more metrics listed with the lmi:metric property. These will be passed to the Luzzu API to start a Quality Assessment. Metrics should exist in the Luzzu framework prior to invocation.

Described by:

lmi:metric

Luzzu After Arg

Luzzu Before Arg

Specifies before arguments for Complex Metrics

Luzzu Metric Arg

This is an abstract class that is extended by lmi:LuzzuBeforeArg and lmi:LuzzuAfterArg. This should not be used directly.

Described by:

lmi:parameter , lmi:type

Luzzu Metric Java Implementation

Registers a Java metric implementation as a Luzzu plugin.
